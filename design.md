# 퓨처에너지 난방케어 - 디자인 계획

## 앱 개요
지역난방 전문업체 고객용 모바일 앱. 40대 이상 고객도 쉽게 사용할 수 있도록 큰 글씨, 단순하고 깔끔한 디자인을 지향한다.

## 화면 목록

### 고객용 화면
1. **홈 화면** (`/`) - 4개 주요 기능 버튼 + 앱 헤더
2. **난방 고장 접수** (`/report`) - 접수 폼 (이름, 연락처, 주소, 증상, 사진, 방문 희망일)
3. **배관청소 신청** (`/pipe-cleaning`) - 배관청소 전용 접수 폼
4. **방문 예약 확인** (`/reservation`) - 접수번호로 예약 조회
5. **점검 결과 확인** (`/inspection-result`) - 접수번호로 처리 결과 조회

### 관리자 화면
6. **관리자 로그인** (`/admin/login`) - 관리자 비밀번호 입력
7. **관리자 접수 목록** (`/admin`) - 전체 접수 목록, 상태별 필터
8. **접수 상세 / 처리** (`/admin/[id]`) - 상세 정보, 기사 배정, 상태 변경

---

## 주요 사용자 흐름

### 고장 접수 흐름
홈 → "난방 고장 접수" 버튼 → 폼 입력 → 사진 첨부 → 방문 희망일 선택 → 제출 → 접수번호 확인 화면

### 예약 확인 흐름
홈 → "방문 예약 확인" 버튼 → 접수번호/전화번호 입력 → 예약 정보 표시

### 관리자 처리 흐름
관리자 로그인 → 접수 목록 → 상세 보기 → 기사 배정 → 상태 변경

## 레이아웃 원칙

- **최소 폰트 크기**: 16px (본문), 20px (제목), 24px (대형 버튼)
- **버튼 높이**: 최소 60px (손가락으로 쉽게 탭 가능)
- **여백**: 넉넉한 패딩 (px-5, py-4 이상)
- **아이콘**: 의미를 직관적으로 전달하는 아이콘 + 텍스트 병용
- **색상 대비**: 배경과 텍스트 간 충분한 대비 확보
- **탭 바**: 홈, 접수, 예약확인, 관리자 4개 탭

무엇을 확인할 수 있나요?

퓨처에너지테크 난방 상태 센서가 설치되면 아래 정보를 앱과 홈페이지에서 실시간으로 확인할 수 있습니다.

💧

세대별 난방 유량 확인

현재 유량(LPM)과 기준 유량을 비교하여 난방수가 정상적으로 공급되고 있는지 확인합니다. 기준 유량에서 크게 벗어나면 즉시 알림을 드립니다.

📊

공급측·환수측 압력 확인

공급측 압력과 환수측 압력, 차압을 실시간으로 모니터링합니다. 압력 이상은 배관 막힘이나 누수의 초기 신호일 수 있습니다.

🚨

이상 알림 자동 발송

기준 유량에서 크게 벗어난 상태가 일정 시간 이상 지속되면 담당 기사와 본사 관리자에게 자동으로 알림이 발송됩니다.

🟢

정상 / 주의 / 점검 필요 표시

복잡한 수치 대신 정상·주의·점검 필요 세 가지 상태로 쉽게 확인할 수 있습니다. 점검 필요 상태에서는 즉시 점검 요청 버튼이 활성화됩니다.

📡

통신 상태 및 마지막 측정 시간

센서와 서버 간 통신이 정상인지 확인하고, 마지막으로 데이터가 수신된 시간을 표시합니다. 통신 두절 시에도 알림을 드립니다.

🔧

원터치 점검 요청

이상 상태 확인 즉시 앱 또는 홈페이지에서 점검 요청 버튼 하나로 담당 지사에 접수됩니다. 별도 전화 없이 빠르게 처리됩니다.
